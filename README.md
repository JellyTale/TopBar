# TopBar
A lightweight desktop utility that adds a macOS-style menu bar to Windows.

Official pre-built binaries are available for purchase over at https://iamboingus.itch.io/topbar for convenience (no build hassle). The full source is here under GPLv3 so you can compile it yourself for free if you prefer.
